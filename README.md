# The movie Database

A place where you can browse through and search for your favourite movies, tv shows.

## Getting Started

In the project directory, you can run:

### `npm start`

Run the app in development mode.

### `npm run build`

Build app for production to the `build` folder.

## Technologies Used

- ReactJS
- Sass
- HTML5
- TMDb API
